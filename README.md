# SEO 키워드 생성기

국내 장기렌트·월렌트 시장을 타겟으로 한 SEO 키워드 조합 자동화 도구입니다. 기존에 스프레드시트에 수동으로 키워드 조합을 만들던 작업을 자동화하기 위해 만들었어요.

---

## 이게 왜 필요한가

장기렌트 광고를 집행할 때 네이버 검색광고나 구글 광고에 넣을 키워드를 직접 만들다 보면 조합이 끝이 없습니다. "신용불량 장기렌트 서울 강남 SUV 그랜저"처럼 여러 속성을 붙여 넣는 롱테일 키워드 특성상, 손으로 만들면 빠뜨리거나 중복이 생기기 마련이고요.

7개 카테고리(기본 키워드, 광역 지역, 세부 지역, 렌탈 상품, 차급, 차종, 고객 유형)를 조합해서 최대 15,000개까지 한 번에 만들고 엑셀로 뽑아갈 수 있게 했습니다.

---

## 기능

- **키워드 생성** — 원하는 수량(100~15,000개)만큼 랜덤 조합으로 생성
- **필터 설정** — 7개 카테고리별로 원하는 옵션만 선택해서 범위 좁히기 가능
- **결과 검색** — 생성된 키워드 내에서 특정 단어로 빠르게 필터링
- **엑셀 다운로드** — 키워드 조합 시트 + 필터 옵션 참조 시트 2개로 구성된 `.xlsx` 파일 내보내기
- **통계 대시보드** — 이론상 가능한 전체 조합 수, 생성 수, 필터링 수 한눈에 확인
- **다크 모드** — `d` 키로 토글 가능

## 기술 스택

| 구분 | 내용 |
|------|------|
| 프레임워크 | Next.js 15 + React 19 + TypeScript |

## 로컬 실행

### 사전 준비

- Node.js 18 이상 (개발 환경 기준 v24 사용)
- npm

### 설치 및 실행

```bash
git clone <repo-url>
cd seo-keyword-generator

npm install
npm run dev
```

브라우저에서 [http://localhost:3000](http://localhost:3000) 열면 됩니다.

### 빌드 및 프로덕션 실행

```bash
npm run build
npm start
```

### 기타 명령어

```bash
npm run lint     # ESLint 검사
```

## 파이썬 스크립트

UI 없이 바로 15,000개짜리 엑셀 파일이 필요하다면 스크립트를 직접 실행해도 됩니다.

```bash
pip install pandas openpyxl
python scripts/generate_keywords.py
```

`seo_keywords_15000.xlsx` 파일이 생성됩니다.

## 키워드 카테고리 구성

| 카테고리 | 항목 수 | 예시 |
|----------|---------|------|
| 기본 키워드 | 9개 | 월렌트, 무심사 장기렌트, 단기렌트 |
| 광역 지역 | 10개 | 서울, 경기, 인천 |
| 세부 지역 | 50개+ | 강남, 송파, 수원 |
| 렌탈 상품 | 14개 | 1개월 렌트, 12개월 렌트 |
| 차급 | 10개 | 경차, 중형차, SUV |
| 차종 | 47개 | K5, 쏘나타, 아반떼 |
| 고객 유형 | 18개 | 신용불량, 개인회생, 외국인 |

7개 카테고리를 모두 곱하면 이론상 수백만 개의 조합이 가능합니다.
